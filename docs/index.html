Features

Everything you need,
intelligently integrated

From natural language task creation to meeting transcription, TaskPilot brings AI into every part of your workflow.

AI Chat Assistant

Manage tasks through natural language. Say "create a task for the API review due Friday" and it's done. Streaming responses with tool-use capabilities.

Smart Suggestions

Daily summaries, weekly reports, task decomposition, priority optimization, and auto-tagging — all powered by Claude AI.

Feishu Integration

Bidirectional sync with Feishu Bitable. Bot notifications for task changes. Seamless team collaboration across platforms.

Meeting Recording

Record meetings, transcribe with Whisper, identify speakers, and let AI extract action items and create tasks automatically.

Command Palette

Fuzzy-search command launcher with customizable keyboard shortcuts. Navigate and act without ever touching the mouse.

Multi-Window

Quick-add floating window, standalone AI chat, and system tray support. A native desktop experience that stays out of your way.

Get Started

Up and running in minutes

Clone, build, and launch — it's that simple.

1

Clone the repository

git clone https://github.com/Davied-H/taskpilot.git
cd taskpilot
2

Install dependencies

Make sure you have Go 1.25+, Node.js 18+, and Wails v3 installed.

go install github.com/wailsapp/wails/v3/cmd/wails3@latest
cd frontend && npm install && cd ..
3

Run in development mode

wails3 dev
